# Employee Attendance System

This project is a web-based application used to manage employee attendance efficiently.

## Technologies Used
- Python (Flask)
- HTML
- CSS
- Bootstrap
- SQLite

## Features
- Employee login
- Mark daily attendance
- View attendance records
- Admin dashboard

## How to Run
1. Install Python
2. Install Flask
3. Run `python app.py`
4. Open browser and go to `http://127.0.0.1:5000`
